## 1. 登录模块 (Login)

### 1.1 组件作用
登录模块是用户进入系统的入口，负责验证用户身份并管理用户会话状态。

### 1.2 实现逻辑

#### 1.2.1 数据定义
```javascript
// 表单数据
const formData = reactive({
  username: '',  // 用户名或邮箱
  password: ''   // 密码
})

// 验证规则
const rules = reactive({
  username: [
    { required: true, message: '请输入用户名', trigger: 'blur' }
  ],
  password: [
    { required: true, message: '请输入密码', trigger: 'blur' }
  ]
})
```

#### 1.2.2 登录流程
1. **用户输入**：用户在表单中输入用户名和密码
2. **表单验证**：点击登录按钮后，触发`submitForm`函数，首先进行表单验证
3. **API调用**：验证通过后，调用`login` API发送登录请求
4. **Token存储**：登录成功后，将返回的token和用户信息存储到localStorage
5. **页面跳转**：根据用户角色（userType）跳转到不同页面

#### 1.2.3 关键函数
- `submitForm(formEl)`: 处理登录表单提交
  - 参数：formEl - 表单引用
  - 流程：验证表单 → 调用登录API → 存储token → 跳转页面

## 10. Markdown渲染器 (MarkdownRenderer)

### 10.1 组件作用
MarkdownRenderer组件用于渲染Markdown格式的内容，支持代码块、粗体、斜体、标题、链接等Markdown语法。

### 10.2 实现逻辑

#### 10.2.1 数据处理
```javascript
// 简单的Markdown渲染器
const renderedContent = computed(() => {
  let html = props.content

  // 转义HTML标签（防止XSS）
  html = html.replace(/</g, '&lt;').replace(/>/g, '&gt;')

  // 处理代码块（```）
  html = html.replace(/```(\w+)?\n([\s\S]*?)\n```/g, (match, lang, code) => {
    return `<pre class="code-block"><code class="language-${lang || 'text'}">${code.trim()}</code></pre>`
  })

  // 处理行内代码（`）
  html = html.replace(/`([^`]+)`/g, '<code class="inline-code">$1</code>')

  // 处理粗体（**）
  html = html.replace(/\*\*(.*?)\*\*/g, '<strong>$1</strong>')

  // 处理斜体（*）
  html = html.replace(/\*(.*?)\*/g, '<em>$1</em>')

  // 处理标题
  html = html.replace(/^### (.*$)/gm, '<h3>$1</h3>')
  html = html.replace(/^## (.*$)/gm, '<h2>$1</h2>')
  html = html.replace(/^# (.*$)/gm, '<h1>$1</h1>')

  // 处理链接
  html = html.replace(/\[([^\]]+)\]\(([^)]+)\)/g, '<a href="$2" target="_blank" rel="noopener noreferrer">$1</a>')

  // 处理无序列表
  html = html.replace(/^- (.*)$/gm, '<li>$1</li>')
  html = html.replace(/(<li>.*<\/li>)/s, '<ul>$1</ul>')

  // 处理有序列表
  html = html.replace(/^\d+\. (.*)$/gm, '<li>$1</li>')

  // 处理引用
  html = html.replace(/^> (.*)$/gm, '<blockquote>$1</blockquote>')

  // 处理分割线
  html = html.replace(/^---$/gm, '<hr>')

  // 处理换行
  html = html.replace(/\n/g, '<br>')

  // 清理多余的br标签
  html = html.replace(/<br><br>/g, '<br>')

  return html
})
```

#### 10.2.2 渲染流程
1. **接收内容**：接收Markdown格式的内容作为props
2. **内容处理**：使用正则表达式处理各种Markdown语法
3. **HTML转换**：将Markdown转换为HTML格式
4. **安全处理**：转义HTML标签，防止XSS攻击
5. **样式应用**：应用相应的CSS样式

#### 13.2.2 请求拦截器
```javascript
service.interceptors.request.use(
  config => {
    // 获取token并添加到请求头
    const token = localStorage.getItem('token')
    if (token) {
      config.headers['token'] = token
    }
    return config
  },
  error => {
    return Promise.reject(error)
  }
)
```

#### 13.2.3 响应拦截器
```javascript
service.interceptors.response.use(
  response => {
    const { data, config } = response
    
    // 业务状态码判断
    if (data.code === '200') {
      return data.data
    } else {
      if (data.code === '-1') {
        // 请求超时，需要重新登录
        if (!config.url?.includes('/login')) {
          ElMessage.error(data.msg || '请求超时，请重新登录')
          // 清除登录信息
          localStorage.removeItem('token')
          localStorage.removeItem('userInfo')
          // 跳转到登录页
          window.location.href = '/auth/login'
        }
      } else {
        // 其他错误
        ElMessage.error(data.msg || '网络请求失败')
        return Promise.reject('网络请求失败')
      }
    }
  },
  error => {
    return Promise.reject(error)
  }
)
```
